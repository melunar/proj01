  • Safari 和 Chrome -webkit-
  • <>IE 9 -ms-

border-radius

border-radius 属性允许您向元素添加圆角。 


div {
    text-align:center;
    border:2px solid #a1a1a1;
    padding:10px 40px;
    background:#dddddd;
    width:350px;
    border-radius:25px;
    -moz-border-radius:25px;/* 老的 Firefox */s
}


box-shadow 用于向方框添加阴影



div {
width:300px;
height:100px;
background-color:#ff9900;
-moz-box-shadow: 20px 10px 5px #342338; /* 老的 Firefox */
box-shadow: 20px 10px 5px #123456;
}

（了解）border-image 使用图片创建围绕 div 元素的边框


div {
-moz-border-image:url(/i/border.png) 30 30 round/stretch;	/* Old Firefox */
-webkit-border-image:url(/i/border.png) 30 30 round/stretch;	/* Safari and Chrome */
-o-border-image:url(/i/border.png) 30 30 round/stretch;		/* Opera */
border-image:url(/i/border.png) 30 30 round/stretch;
}
